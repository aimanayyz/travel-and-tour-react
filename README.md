# 🌍 TravelX – Tour & Travel Website

TravelX is a modern, responsive Tour & Travel website built using **React**, **Vite**, and **Tailwind CSS**. It showcases popular travel destinations, tour packages, services, testimonials, and a contact section in a clean single-page layout.

## ✨ Features

- 🏠 Responsive Navigation Bar
- 🌄 Hero Section
- 🌍 Popular Destinations
- ✈️ Travel Services
- 🎒 Tour Packages
- 💬 Customer Testimonials
- 📞 Contact Form
- 📱 Mobile-Friendly Design
- ⚡ Fast Performance with Vite
- ♻️ Reusable React Components
- 🎨 Styled with Tailwind CSS

## 🛠️ Technologies Used

- React
- Vite
- Tailwind CSS
- React Icons
- JavaScript (ES6+)
- HTML5
- CSS3

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/travel-and-tour-react.git
```

Navigate to the project directory:

```bash
cd travel-and-tour-react
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:5173
```
